# Ola Ride Data Analysis

## Project Overview

This project analyzes Ola ride booking data using **Excel**, **SQL**, and **Power BI** to uncover business insights related to booking trends, cancellations, revenue, customer behavior, and vehicle performance.

The project demonstrates the complete data analytics workflow—from data cleaning and preprocessing to SQL analysis and interactive dashboard creation.

---

## Tech Stack

- Microsoft Excel
- MySQL
- Power BI
- DAX

---

## Project Workflow

### 1. Data Cleaning (Excel)

- Removed duplicate records
- Handled missing values
- Standardized data formats
- Verified data consistency
- Prepared the dataset for analysis

### 2. SQL Analysis

Performed business analysis using SQL:

- Total Bookings
- Successful Bookings
- Cancelled Bookings
- Revenue Analysis
- Vehicle-wise Performance
- Customer vs Driver Cancellations
- Average Ride Distance
- Payment Method Analysis
- Ratings Analysis

### 3. Power BI Dashboard

Created an interactive dashboard featuring:

- KPI Cards
- Booking Trends
- Revenue Analysis
- Cancellation Rate
- Vehicle Type Analysis
- Payment Distribution
- Customer & Driver Ratings
- Interactive Filters (Slicers)
